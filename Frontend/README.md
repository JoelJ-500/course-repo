# CourseRepo Wireframe UI (Static HTML/CSS)

This folder contains static HTML pages styled to match the provided wireframes.

## Pages
- `index.html` — Home
- `search.html` — Search results + filters
- `course.html` — Course page + upload modal
- `add-course.html` — Add new course form
- `saved.html` — My saved content
- `login.html` — Login
- `signup.html` — Registration (linked from login)
- `upload.html` — Standalone upload dialog (with Description)
- `upload-misc.html` — Standalone upload dialog (no Description)

## Run locally
Run with a http server. For example:
1. Install using `npm install http-server -g`
2. Run with `http-server`
