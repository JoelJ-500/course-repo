# course-repo
Web platform for students to share, discover, and prepare with community-uploaded university course materials.

## MVP User Stories

### Students (Primary Users)

- **As a student**, I want to search for a course by university, course code, or professor so I can quickly find study materials without relying on group chats or forums.
- **As a student**, I want to view all available materials for a course in one place so I can prepare efficiently for exams and assignments.
- **As a student**, I want to upload past exams, assignments, or outlines so future students can benefit from what I already have.
- **As a student**, I want to create a course page if it doesn’t exist so useful material isn’t lost or inaccessible.
- **As a student**, I want to see basic course details (difficulty, professor, last updated) so I know what to expect before taking the course.

### Professors

- **As a professor**, I want to view previous course content so I can understand how the course was taught before and improve my version.

### University Administrators

- **As a university administrator**, I want to compare similar courses across universities so I can evaluate and improve curriculum quality.

### Prospective Students

- **As a prospective student**, I want to view real course materials so I can decide whether a course or program is right for me.
