# course-repo
Web platform for students to share, discover, and prepare with community-uploaded university course materials.

## User Stories

| Story ID | Story Title        | User Story |
|--------|-------------------|------------|
| ST-1 | Course Search | As a student, I want to **search for a course** by university, course code, or professor so I can quickly find study materials without relying on group chats or forums |
| ST-2 | Material Display | As a student, I want to **view all available materials for a course** on a page so I can prepare efficiently for exams and assignments. |
| ST-3 | Course Metadata | As a student, I want to **see basic course details (difficulty, professor, last updated)** so I know what to expect before taking the course. |
| ST-4 | Upload files | As a student, I want to **upload past exams, assignments, or outlines** so future students can benefit from what I already have. |
| ST-5 | Create course | As a student, I want to **create a course page** if it doesn’t exist so useful material isn’t lost or inaccessible. |
| ST-6 | Course Preview | As a prospective student, I want to **view real course materials** so I can decide whether a course or program is right for me. |
| PROF-1 | Content Review | As a professor, I want to **view previous course content** so I can understand how the course was taught before and improve my version. |
| UNI-1 | Course Compare | As a university administrator, I want to **compare similar courses across universities** so I can evaluate and improve curriculum quality. |
